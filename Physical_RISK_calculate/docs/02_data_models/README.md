# 데이터 모델

이 폴더에는 프로젝트의 데이터베이스 스키마를 정의한 DBML 파일들이 포함되어 있습니다.

## 📊 DBML 파일 목록

### 주요 스키마

- **ERD_Consolidated_v1.0.dbml** - 통합 ERD v1.0
- **complete_data.dbml** - 전체 데이터 스키마
- **local_data.dbml** - 로컬 데이터 스키마
- **physical_risk_api_erd.dbml** - Physical Risk API ERD
- **ssp.dbml** - SSP 시나리오 데이터 스키마

## 🛠️ 사용 방법

DBML 파일은 [dbdiagram.io](https://dbdiagram.io)에서 시각화할 수 있습니다.
