## 💡 Vulnerability 산출 활용

### 수자원 의존도 취약성 평가

- **목적**: 지역별 지하수 의존도 및 가용성 평가
- **활용 방법**:
  1. 사업장 위치의 지역 지하수 이용 현황 조회
  2. 용도별 이용량 분석
  3. 높은 의존도 + 제한적 수자원 → 가뭄 시 취약성 증가

### 리스크별 활용

| 리스크                     | 활용 방법                                 |
| -------------------------- | ----------------------------------------- |
| **Water Stress (물 부족)** | 지하수 의존도 → 가뭄 시 공급 중단 리스크  |
| **Extreme Heat (폭염)**    | 생활용수/공업용수 수요 급증 → 지하수 고갈 |
| **Extreme Cold (한파)**    | 융설용 지하수 수요 → 겨울철 부하          |

### 계산 지표

- **지하수 의존도**: 총 이용량 / 지역 인구
- **용도별 비율**: 생활용 vs 공업용 vs 농업용 비중
- **개소수 밀도**: 개소수 / 지역 면적

## ⚠️ 주의사항

1. **데이터 갱신 주기**:

   - 지하수 조사연보 기반 (연 1회 통계)
   - 실시간 데이터 아님
   - 최신 연도 데이터 확인 필요

2. **지역 코드**:

   - `ctpv`, `sgg` 파라미터는 정확한 지역명 입력 필요
   - 시도/시군구 표준 명칭 사용
   - 사업장 위치와 매칭 로직 필요

3. **용도 구분**:

   - `plcCo1~8`, `utztnQy1~8` 필드는 API별로 의미가 다름
   - 각 엔드포인트의 용도 매핑 확인 필수
   - 기타용(`etc`)은 미분류 항목

4. **이용량 단위**:

   - 모든 이용량은 ㎥/년 단위
   - 일평균 환산 시 365로 나누기
   - 계절별 변동 고려 안 됨

5. **API 호출**:
   - 필수 파라미터 누락 시 오류
   - 페이징 처리 권장
   - 전국 조회 시 시도/시군구 생략 가능

## 🎯 V(Vulnerability) 계산 활용

이 API는 **Water Stress (물 부족)** 리스크의 **V(Vulnerability)** 계산에 활용됩니다.

### 취약성 점수 산출

- **높은 지하수 의존도** → 가뭄 시 대체 수원 부족 → 취약성 ↑
- **높은 개소수 밀도** → 지하수 과다 개발 → 고갈 위험 ↑
- **특정 용도 편중** → 다변화 부족 → 유연성 ↓

### 지역별 상대 평가

```
취약성 점수 = (정규화된 1인당 이용량) × 0.4
           + (정규화된 개소수 밀도) × 0.3
           + (용도 편중도) × 0.3

용도 편중도 = 1 - (생활용 비율 × 공업용 비율 × 농업용 비율)^(1/3)
```
