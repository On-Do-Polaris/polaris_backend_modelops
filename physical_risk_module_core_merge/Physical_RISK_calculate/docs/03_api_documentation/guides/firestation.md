# 소방청 전국소방서 좌표 현황 API

## 1. 개요
- **API:** 소방청_전국소방서 좌표현황(XY좌표)
- **제공기관:** 소방청
- **설명:** 전국의 소방서 및 119안전센터의 명칭, 주소, 전화번호, 그리고 XY좌표(위경도) 정보를 제공하는 API입니다. 이 데이터를 사용하여 특정 위치로부터 가장 가까운 소방서까지의 거리를 계산하고, 이를 통해 소방 접근성을 평가할 수 있습니다.

## 2. 요청 정보
- **Base URL:** `https://api.odcloud.kr/api`
- **인증:**
  - **인증키:** 필요 (공공데이터포털 'odcloud'에서 발급)
  - **전달 방식:** API URL에 인증키가 포함된 고유 식별자(uddi)를 사용하거나, 별도의 인증 파라미터로 전달할 수 있습니다. (문서에 따라 다름)
- **데이터 포맷:** JSON, XML (요청 파라미터 `returnType`으로 지정, 기본값: JSON)

## 3. 주요 오퍼레이션

### 가. 전국소방서 좌표현황 조회
- **Endpoint:** `/15138232/v1/uddi:da0c6c93-f05a-453d-849f-e4c3697222e3`
- **Method:** `GET`
- **설명:** 페이징 처리를 통해 전국의 소방서 및 안전센터 목록과 관련 정보를 조회합니다.
- **요청 파라미터:**
  | 이름 | 타입 | 필수 | 기본값 | 설명 |
  |---|---|---|---|---|
  | `page` | integer | N | 1 | 페이지 번호 |
  | `perPage` | integer | N | 10 | 한 페이지에 표시할 결과 수 |
  | `returnType` | string | N | JSON | 응답 데이터 포맷 (`JSON` 또는 `XML`) |

- **주요 응답 필드:**
  - `소방서 및 안전센터명`: 소방서 또는 119안전센터의 명칭
  - `주소`: 상세 주소
  - `X좌표`: 경도 (Longitude)
  - `Y좌표`: 위도 (Latitude)
  - `유형`: 기관 유형 (예: 소방서, 119안전센터)
