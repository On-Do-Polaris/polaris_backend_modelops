# 산림청 전국 산불위험지역 표준 데이터 API

## 1. 개요
- **API:** 전국산불위험지역표준데이터
- **제공기관:** 산림청
- **설명:** 지형, 임상, 기상 자료를 바탕으로 전국 행정구역(시/군/구) 단위의 시간대별 산불위험 예보 정보를 제공하는 API입니다. 산불위험지수는 등급(낮음~매우높음)으로 산출되며, 예방 및 대응 활동에 활용됩니다.

## 2. 요청 정보
- **Base URL / Endpoint:** `http://apis.data.go.kr/1400377/forestPoint/forestPointListGeongugSearch`
- **인증:**
  - **인증키:** 필요 (공공데이터포털에서 발급)
  - **전달 방식:** 요청 파라미터 `ServiceKey`에 포함하여 전송
- **데이터 포맷:** XML, JSON (요청 파라미터 `_type`으로 지정)

## 3. 주요 오퍼레이션

### 가. 전국 산불위험지수 조회
- **Method:** `GET`
- **설명:** 특정 시점의 전국 산불위험지수 예보 정보를 조회합니다. 3시간 간격으로 72시간까지의 예측 정보를 제공합니다.
- **요청 파라미터:**
  | 이름 | 파라미터명 | 타입 | 필수 | 설명 |
  |---|---|---|---|---|
  | 서비스키 | `ServiceKey` | string | Y | 공공데이터포털에서 발급받은 인증키 |
  | 데이터 타입 | `_type` | string | N | 응답 데이터 포맷 (`json` 또는 `xml`) |
  | 페이지 번호 | `pageNo` | integer | N | 페이지 번호 |
  | 항목 수 | `numOfRows` | integer | N | 한 페이지에 표시할 결과 수 |
  | 예보 제외 여부 | `excludeForecast` | string | N | 예보 정보 제외 여부 (1:제외, 0:포함) |

- **주요 응답 필드:**
  - `analdate`: 데이터 기준 일자
  - `doname`: 시도 명칭
  - `regioncode`: 지역 코드
  - `maxi`: 산불위험예보지수 최댓값
  - `meanavg`: 산불위험예보지수 평균
  - `d1` ~ `d4`: 위험 등급별(낮음~매우높음) 면적 또는 비율
