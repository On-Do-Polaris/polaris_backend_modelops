# H (Hazard) 시나리오 데이터 설명

## 1. 개요

**H (Hazard, 위해성) 점수**는 물리적 리스크 모델(`H × E × V`)의 첫 번째 요소로, 미래 기후 변화 시나리오에 따른 순수 기후 위험의 강도와 빈도를 나타냅니다.

이 데이터는 특정 지역이나 사업장이 겪게 될 기후 현상 그 자체의 위험 수준을 정량화한 값입니다.

---

## 2. 데이터 소스: CMIP6 SSP 시나리오

본 프로젝트에서 사용하는 H-Score는 **CMIP6(Coupled Model Intercomparison Project Phase 6)** 데이터셋을 기반으로 산출되었습니다. CMIP6는 IPCC 제6차 평가 보고서(AR6)의 기반이 된 최신 기후 모델링 데이터로, 전 세계 여러 연구기관의 기후 예측 모델 결과를 포함합니다.

이를 통해 미래의 다양한 사회경제적 경로(SSP)에 따라 기후가 어떻게 변할지를 예측하고, 이를 바탕으로 각 물리적 리스크의 위험도를 계산합니다.

---

## 3. SSP(공통사회경제경로) 시나리오 종류

본 프로젝트는 4가지 주요 SSP 시나리오를 사용하며, 각 시나리오는 미래의 탄소 배출 수준과 사회경제적 변화를 나타냅니다.

| 시나리오     | 설명                                            | 특징                                                                                                         |
| :----------- | :---------------------------------------------- | :----------------------------------------------------------------------------------------------------------- |
| **SSP1-2.6** | **지속가능한 발전 경로 (저탄소 시나리오)**      | 2100년까지 지구 평균 온도 상승을 1.5°C 이내로 제한하는 것을 목표로 하는 가장 낙관적인 시나리오입니다.        |
| **SSP2-4.5** | **중간 경로 (중도 시나리오)**                   | 현재와 유사한 수준의 사회경제적 발전과 기후 정책이 유지될 경우를 가정한 시나리오입니다.                      |
| **SSP3-7.0** | **지역적 경쟁 심화 경로 (고탄소 시나리오)**     | 국가 간 협력이 감소하고 각자도생하는 상황에서 기후 대응 노력이 미미할 경우를 가정한 비관적인 시나리오입니다. |
| **SSP5-8.5** | **화석연료 기반 발전 경로 (초고탄소 시나리오)** | 기술 발전은 빠르지만 화석연료에 대한 의존도가 매우 높고, 적극적인 기후 정책이 부재한 최악의 시나리오입니다.  |

---

## 4. 데이터 파일 종류

`physical_risks/` 디렉토리에는 각 시나리오와 리스크별로 두 가지 종류의 파일이 제공됩니다.

- **`.nc` (NetCDF)**

  - **형식**: Gridded (격자형) 데이터
  - **내용**: 위도와 경도 좌표에 따른 상세한 리스크 점수가 포함된 원시 데이터 파일입니다. 특정 좌표의 정확한 H-Score를 조회할 때 사용됩니다.
  - **예시**: `SSP126_extreme_heat_gridded.nc`

- **`.csv` (CSV)**
  - **형식**: Table (테이블) 데이터
  - **내용**: 특정 지역이나 자산별로 집계된 리스크 점수가 포함된 파일입니다. 전체적인 리스크 분포를 확인하거나 요약된 정보를 사용할 때 유용합니다.
  - **예시**: `SSP126_extreme_heat.csv`

---

## 5. 제공되는 리스크 종류

아래 9대 물리적 리스크에 대한 H-Score 데이터가 각 SSP 시나리오별로 제공됩니다.

1.  **극심한 고온 (Extreme Heat)**
2.  **극심한 한파 (Extreme Cold)**
3.  **산불 (Wildfire)**
4.  **도시 홍수 (Pluvial Flooding)**
5.  **내륙홍수 (Fluvial Flooding)**
6.  **해안홍수 (Sea-Level Rise)**
7.  **가뭄 (Drought)**
8.  **물부족 (Water Stress)**
9.  **태풍 (Tropical Cyclone)**

---

## 6. 데이터 위치

모든 H-Score 관련 데이터 파일은 프로젝트 루트의 `physical_risks/` 디렉토리 내에 위치합니다.
